该项目最主要的功能是提供一个 todo 列表，用户可以在列表中添加、删除、修改 todo 项。
最后的效果是打包成一个 iOS 的 app 软件，用户可以在 iOS 设备上安装并使用。
前端的界面和组件主要包括：
- 登录页 (/login) ：输入邮箱与密码；登录后进入主界面，并请求通知权限。
- 主界面 (/app) ：任务列表 + 新建任务表单（含截止时间、到期提醒、预提醒）。
- 设置页 (/settings) ：查看/请求通知权限，退出登录，返回主界面。
- 注册页 (/register) ：输入邮箱、密码、确认密码；注册成功后进入登录页。
- 添加 todo 项：用户可以在主界面上添加新的 todo 项。
- 删除 todo 项：用户可以在主界面上删除已完成的 todo 项。
- 修改 todo 项：用户可以在主界面上修改未完成的 todo 项。
前端的界面和组件主要参考https://reactbits.dev/中的样式，在此基础上进行修改
